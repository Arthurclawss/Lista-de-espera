# Lista de Espera por Dia

Um aplicativo web simples para gerenciamento de listas de espera por dia da semana. Ideal para pequenas empresas ou serviços que precisam organizar atendimentos em horários específicos.

---

## 🌟 Funcionalidades

- Gerenciamento de listas de espera para cada dia da semana (Segunda a Sábado).  
- Adição de entradas com **horário** e **nome do cliente**.  
- Validação de horário (apenas entre **07:30 e 18:00**).  
- Remoção fácil de entradas com botão ❌.  
- Layout moderno e responsivo usando **CSS Grid**.  

---

## 💻 Tecnologias

- HTML5  
- CSS3  
- JavaScript (vanilla)  

---

## ⚡ Como usar

1. Abra o arquivo `index.html` em qualquer navegador moderno.  
2. Clique no botão **Adicionar** abaixo do dia desejado.  
3. Preencha o **horário** e o **nome** nos prompts que aparecerem.  
4. Para remover uma entrada, clique no botão ❌ ao lado do nome.  

> ⚠️ Os horários válidos são entre 07:30 e 18:00.
